✅ JSON Array

  • What it is: A data structure in JSON that holds an ordered list of values.
  • Syntax: Square brackets []
  • Example: 
    [
  "apple",
  "banana",
  "cherry"
]
  • Values can be:
    • Strings ("apple")
    • Numbers (42)
    • Booleans (true)
    • Null (null)
    • Objects ({ "name": "John" })
    • Other arrays

✅ List

  • General term for an ordered collection of items.
  • Used in many programming languages:
    • Python: list
    • JavaScript: Array
    • Java: List<E>
    • JSON: No explicit term "list", but arrays serve this role
